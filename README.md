## PATTERN (using loops)

[CODE](/pattern.c)

```
STEP 1  :   START
STEP 2  :   SET I = 1
STEP 3  :   SET J = 1
STEP 4  :   PRINT '*\t'
STEP 5  :   SET J = J + 1
STEP 6  :   IF J <= I
STEP 7  :       GOTO 4
STEP 8  :   PRINT '\n'
STEP 9  :   SET I = I + 1 
STEP 10 :   IF I <= 4
STEP 11 :       GOTO 3
STEP 12 :   STOP
```

## AREA AND CIRCUMFERENCE

[CODE](/circle.c)

```
STEP 1  :   START
STEP 2  :   READ R
STEP 3  :   CALCULATE A = 3.14 * R * R
STEP 4  :   CALCULATE C = 2 * 3.14 * R
STEP 5  :   PRINT A , C
STEP 6  :   STOP
```

## SIMPLE INTEREST
